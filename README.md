# TRIBE v2 Dashboard Live

Public static build of the TRIBE v2 brain dashboard prototype.

Live site:

```text
https://avi09cr7.github.io/tribev2-dashboard-live/
```

This version runs fully in the browser. Uploading a video stays local to the browser session, and the dashboard derives realtime graph and engagement signals from browser-side video/audio sampling.

This public repo does not include the private TRIBE source checkout, local conda environment, generated prediction files, or model weights. Real TRIBE inference will need a backend API or precomputed outputs in a later version.
